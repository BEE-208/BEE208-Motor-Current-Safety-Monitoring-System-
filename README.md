# Motor Current Safety Monitoring System

## Project Overview

The Motor Current Safety Monitoring System is a C++ application developed to monitor the operating current of electric motors. It compares the measured current with the rated current and determines whether a motor is operating safely or under overload conditions.

## Features

- Monitor multiple motors
- Input validation
- Automatic overload detection
- Current difference calculation
- Generate motor reports
- Save reports to a text file
- Object-Oriented Programming (OOP)

1. Clone the repository.
2. Open the project in Visual Studio Code or Code::Blocks.
3. Compile and run `main.cpp`.
4. Open `website/index.html` to view the web interface.
