About This System

The Motor Current Safety Monitoring System is a web application designed to monitor the operating condition of electric motors.


How It Works

  • The user enters the motor name.
  • The rated current of the motor is entered.
  • The measured current is entered.
  • The system compares both values.
  • If the measured current is less than or equal to the rated current, the motor is safe.
  • If the measured current exceeds the rated current, the motor is overloaded.

Features

  • Monitor multiple motors (C++ version)
  • Detect overload conditions
  • Calculate current difference
  • Generate reports
  • Simple and user-friendly interface
